In this box

Loris

Loris is a full-featured IIIF image server. It can efficiently serve images at various scales, saving bandwidth for larger images and allowing them to be easily viewed by IIIF viewers such as Mirador and Universal Viewer

UTL IIIF API

The UTL IIIF API brings together the IIIF Presentation API, Image API, Search API along with the IIIF Discovery + Activity Streams into a single tool to work with. This API enables communication between defined IIIF-compliant image stores and standard tools.

Mirador

Mirador is a configurable, extensible, and easy-to-integrate image viewer, which enables image annotation and comparison of images from repositories dispersed around the world. Mirador has been optimized to display resources from repositories that support the IIIF API's.
